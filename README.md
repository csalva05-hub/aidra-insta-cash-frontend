Aidra Frontend - Vite + React + Tailwind

Quick deploy:
1. Create a GitHub repo and upload these files (root).
2. In Vercel, Import Project -> select repo, Framework: Vite.
3. Set environment variable VITE_API_URL to https://aidra-insta-cash-backend.onrender.com/api
4. Deploy. The app will be live and will connect to your backend.

Local test (if you can):
npm install
npm run dev
